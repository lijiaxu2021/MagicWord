## ğŸ“– Introduction

EasyWord is a vocabulary learning application that combines **AI Intelligence** with the **SM-2 Spaced Repetition Algorithm**. It is designed to help users efficiently build their vocabulary and prevent forgetting through scientific review schedules. Version v0.0.4 introduces automatic updates and more user-friendly features.

## âœ¨ Core Features

### 1. Immersive Word Learning (Words Tab)
- **Full-Screen Card Design**: Minimalist style, focusing purely on memorization.
- **Gesture Interaction**: Swipe left/right to switch, click to view detailed analysis.
- **AI Mnemonics**: Provides contextual sentences and root/affix mnemonics generated by AI.

### 2. Intelligent AI Search (Search Tab)
- **Precise Query**: Deep analysis using the Qwen-2.5-7B-Instruct model.
- **One-Click Save**: Search results are directly saved to the current library with automatic deduplication.

### 3. AI Batch Import (Library Tab)
- **Long Text Extraction**: Input articles or paragraphs, and AI automatically extracts new words and phrases.
- **Smart Deduplication**: Automatically filters out existing words.

### 4. Word List (WordList Tab) [New]
- **Custom Lists**: Create personalized word lists (e.g., "TOEFL Core Words").
- **Multi-View Toggle**: Supports List mode and Compact Table mode (view state persisted).
- **Quick Jump**: Double-click a word to jump directly to card learning mode.

### 5. Test & Review (Test Tab)
- **Multiple Choice Mode**: Review tests driven by the SM-2 algorithm.
- **Spelling Mode**: Reinforce spelling memory.
- **Statistics**: Records accuracy and details for each test session.

### 6. System Features [New]
- **Auto Update**: Automatically checks for new GitHub Releases via a reverse proxy node.
- **About Page**: Built-in detailed user guide and version information.
- **Settings Management**: Supports custom AI model parameters and User Persona.
